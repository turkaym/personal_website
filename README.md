# Farid Salomon Portfolio

Portfolio web bilingüe (EN/ES) con foco backend Java.

## Stack
- Next.js + TypeScript
- Tailwind CSS
- Export estático para Hostinger

## Setup local
```bash
npm install
npm run dev
```

## Producción (Hostinger)
```bash
npm run build
```

Subí el contenido de la carpeta `out/` al `public_html/` de Hostinger.

## Pendientes antes de publicar
1. Copiar tu foto en `public/profile.jpg`.
2. Reemplazar `REPLACE_WITH_WEB3FORMS_ACCESS_KEY` en `components/portfolio-page.tsx`.
3. (Opcional) Agregar link de GitHub para El Tano cuando esté disponible.
